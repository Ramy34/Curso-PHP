# Curso Básico de PHP 7

Repositorio con los ejemplos de código y apuntes del **Curso Básico de PHP 7 de Código Facilito**.
En este proyecto se incluyen diversos scripts en PHP que demuestran la sintaxis y características principales del lenguaje.

### Temario
- **Clases 1 a 6:** Introducción, Configuración, Hola Mundo, Variables, Constantes y Conversión de Tipos.
- **Clase 7:** Concatenación e Interpolación (`clase7.php`) - Ejemplos de strings, comillas dobles/simples y caracteres de escape.
- **Clase 8:** Operadores de Comparación (`clase8.php`) - Operadores de igualdad (`==`, `===`), nave espacial (`<=>`), Elvis (`?:`) y fusión null (`??`).
- **Clases 9 y 10:** Operadores Aritméticos y de Asignación.
- **Clase 11:** Operadores Lógicos (`clase11.php`) - Uso de AND (`&&`), OR (`||`), XOR y NOT (`!`).
- **Clase 12:** If, Else, Else if (`clase12.php`) - Estructuras condicionales y sintaxis alternativa con dos puntos (`:`).
- **Clases 13 a 16:** Estructuras de Control (Switch, While, Do-While, For, Foreach).
- **Clase 17:** Array Indexado y Asociativo (`clase17.php`) - Creación, obtención de valores y recorrido de arreglos.
- **Clase 18:** Array Multidimensional (`clase18.php`) - Matrices y arreglos multidimensionales mixtos.
- **Clase 19:** Funciones.
- **Clase 20:** Argumento de Funciones (`clase20.php`) - Parámetros, paso por referencia (`&$num`) y recursividad.
- **Clase 21:** Devolver valores en una función (`clase21.php`) - Valores por defecto, retorno de arreglos y múltiples valores usando `list()`.
- **Clase 22:** Funciones variables y anónimas.
- **Clase 23:** Declaraciones de tipo escalar (`clase23.php`) - Uso de `declare(strict_types = 1)` y tipado estricto.

---
*Nota: Los archivos `.php` incluidos en este repositorio contienen el código práctico documentado para las clases destacadas.*
